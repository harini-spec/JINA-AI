To create and activate new JINA conda environment:  
```
conda create --name JINA    
conda activate JINA   
```
  
    
Install requirements     
```
pip install -r requirements.txt  
conda install -c conda-forge jupyterlab  
```    
  
    
To run streamlit  
```
streamlit run visual_stream.py
```
