To create a new JINA conda environment:  
``` conda create --name JINA```  
Activate Conda  
```conda activate JINA```  


Install requirements  
```pip install -r requirements.txt```  
Install jupyter lab  
```conda install -c conda-forge jupyterlab```  

To run streamlit  
```streamlit run visual_stream.py```
